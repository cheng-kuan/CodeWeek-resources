2016歐洲程式週活動

歐洲程式週活動是由各地推廣程式教育大使們志願性組織的一項草根活動。任何人，包含學校、老師、圖書館、程式社團、企業或公眾機構，都可以組織一場程式週的活動，更能加入歐洲程式週活動的地圖。為了讓策劃和運作活動更簡單，我們準備好了不同的工具，更提供了最好的學習計畫、引導和資源。

給所有年紀新手的程式課程

  • Scratch ode to code: Scratch歐洲程式週各種語言的課程教學
  • Codecademy: 線上免費互動式的程式學習
  • Code School: 程式學校使用影片教學、程式挑戰和螢幕截屏學習網站科技，不同種瀏覽器也能使用
  • Code Avengers: 用HTML、CCS和Javacript學會架設網站、APP和遊戲
  • Code.org Tutorials: 給初學者一小時或更少時間的簡單教學
  • Computer Science Unplugged: 適合教室和家庭的免費整套學習活動，加入卡片、繩子和蠟筆的遊戲與拼圖元素實施程式語言教學
  • Angry Birds Fun Learning: 用好玩的方式學會程式語言！探索各種難度的趣味APP和程式的課程
  • Webmaker Web Literacy Map: 一套完整關於數位技能和網站素養的教學，包含一單元創建網站
  • W3Schools Online Web Tutorials: 一套完整關於網路程式語法的教學和參考資料
  • Academy of Code: 一套互動式HTML、CSS、JAVASCRIPT、PHP和MySQL的組合教學
  • CodinGame: 用程式語言玩電玩遊戲，用超過二十種不同的語法學習程式
  • Silent Teacher: 一步一腳印又好玩的方式學習基礎
  • CodeCombat: 一個學習程式語言的線上遊戲，學生用真實的語法（Python、Javascript、Lua、CofeeScript、Clojure）撰寫程式
  • CodeSpells 精巧地製作屬於你自己的符咒，和世界互動並解決世界的問題
  • CodyRoby: 用不插電的紙牌製作你自己的遊戲和活動
  • Lesson plans for teachers: 為了讓初等和中等教育的老師們學生們體驗程式，而設計出的教學課程，讓學生們用趣味的方式了解程式概念，更提供老師們多種不同的想法和資源
  • Introducing computing in the classroom, by EU Schoolnet: 由老師為老師設計的課程，包含教授、老師、學生和資訊領域專家所提供的訪問、簡報和活動

給年輕新手的程式

  • Scratch: 有了scratch，你可以寫出屬於自己互動式的故事、遊戲和動畫，你更可以和其他人在線上的社群分享。主要是設計給16-18歲學生們探索程式，可以使用不同種的程式語言，更可以用線上或不用連線的Scratch editor（可以用mac,windows或linux電腦），老師們也可以探索教學者們所在的線上社群ScratchED，分享故事、交換資源、詢問問題甚至交朋友
  • Hopscotch: iPad App推薦給八歲以上的孩子，在多彩又互動的環境中，使用簡易和直覺式的磚塊建構出遊戲、動畫和app。如果有更年幼的孩子，可以嘗試Daisy the Dinosaur App
  • ScratchJr: 這是一個程式語言介紹的課程，讓五到七歲的孩子創造自己的互動故事和遊戲。ScratchJr是受Scratch啟發，但重新設計了使用介面和程式語言，讓他們可以發展性適合年幼的孩子。最近可以用iPad app使用，2014年Android version釋出，2015年有了網站架構的版本
  • CoderDojo: 提供家長、孩子和志工多樣化的資訊創建程式教學的社團，網站中有部分和清單的資源，可以根據不同的年紀使用不同種的程式語言教學
  • RoboMind Academy: 當你使用程式語言操控一個機器人，學生會學習到邏輯思考、自動化和科技發展的概念。讓八歲或以上的孩子使用的線上教學平台。可以從一小時程式教學開始
  • Run Marco!: 一個讓孩子學習基礎程式概念的冒險遊戲，可以使用 Android app、網站瀏覽器使用此平台，更有超過13種歐系語言的版本（持續增加中）
  • Pocket Code: 一個可以讓你創建屬於你自己的遊戲、動畫和互動式音樂影片的Android app，還有更多不同種的app，可以直接以手機和電腦使用。一個簡單的方式來開始，Skydiving Steve Skydiving Steve 一小時的程式教學。或是你可以照著Horst的步驟教學將你的手機轉換成國外形式的掃瞄器
  • Kodu: 讓孩子透過簡易視覺性的程式語言來創建遊戲，Kodu可以被拿來教創意力、問題解決、故事敘述力和程式語言。可以以windows電腦使用
  • CodeMonkey: 教你程式語言的有趣線上遊戲，在這個免費科學、技術、工程、數學的教學遊戲中，學生可以學習程式語言概念，像是透過讓猴子找到香蕉，學習程式語言中的功能和迴圈，一個現實㚈使用的程式語言。寫程式，抓香蕉，救世界
  • CodyRoby: 不插電的遊戲和活動
  • iCode from iTeach: 一開始被iTech創建，讓學校在學期間有系統地運作程式語言的社團、課外活動或課程

給進階學習者的完整線上課程

  • edX: 提供知名大學的線上互動課程和MOOCs，MITx、HarvardX、BerkeleyX和UTx 的線上課程
  • MIT OpenCourseWare: 全部MIT課程內文的出版品，OCW是開放又能讓全世界每一個人都享有的資源，而且是永久MIT的活動
  • Coursera: 與全球知名大學和組織合作的教育平台，提供任何人免費的線上課程
  • Udacity: 線上幫助彌平學界和現實生活鴻溝的教育資源，業界領導者像是Google、Facebook、Cloudera和MongoDB都很有熱誠地分享專業能力
